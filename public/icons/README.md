# Playlist Icons Folder

Place your custom playlist icon images here.

## Supported Formats
- PNG (recommended for transparency)
- JPG/JPEG
- SVG
- WebP

## Recommended Size
- 200x200 pixels or larger
- Square aspect ratio (1:1)

## Usage

In `src/data/libraryData.js`, set the icon field:

**For Emoji:**
```javascript
icon: "üôè"
```

**For Custom Image:**
```javascript
icon: "/icons/bhakti-icon.png"
```

## Example File Names
- bhakti-icon.png
- gita-icon.png
- kirtan-icon.png
- festival-icon.png
